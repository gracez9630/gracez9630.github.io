Platformer Game

The Freedom Project for SEP11 is all about platformer games. The purpose of this blog is to understand the process that game developers did when creating games, more specifically platformer games.

For my project, I made a platformer game with my partner Vivian using the tool Kaboom. We studied and used the tool Kaboom JS in order to make my platformer game.

A challenge we faced during this was when our sprites were too big to the point that even if we were to scale it, the sprites would look as if they were floating. This is fixed as we had to crop all the sprites again to have it worked.

Our takeaway for this was that communication is essential in these types of project as if we were not able to communicate with eahc other over a specific set of problems, we could have never overcome those difficulties and could lead to a unsatisfactory freedom project.

For our next step, we wish to atleast use more codes from what we have learned into future projects that we weren't able to use due to the limited amount of time we had left.
